📖 gShell - ReadMe

What is gShell?

gShell is a custom browser-based CLI that lets you interact with Google Sheets like a digital wizard 🧙🏾‍♂️. Whether you’re organizing data, triggering automations, or just flexing terminal aesthetic — this is your command central.

How to Use

  • Type a command like read Sheet1 to fetch data
  • Use the 📊 toggle to switch between Personal and Business sheets
  • Open documentation like this from the side panel

Available Commands

read Sheet1 – pull all rows from a sheet

help – list commands available

clear – wipe terminal log

switch – swap current account context

Notes

gShell uses a serviceAccount.json file for authenticating with the Google Sheets API. Never share this file publicly.
