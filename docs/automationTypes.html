⚙️ gShell Automation Types

AutoSync Mesh

Smart Field Sync

“From auth to sheet, instantly.”

Automatically copies user fields like email or role from your `auth.json` to any sheet with mapping enabled. Set once in schema, and keep your data synchronized as users register.

SheetTriggers

Logic-Driven Actions

“If this... then that — right in your sheet.”

Define data-triggered logic (like: if `role == admin` and `active == false`, then `flagged = true`). Execute internal transformations or set flags automatically as rows update.

gShell Relay

External Messaging

“Trigger emails, alerts, and more.”

Automate communication outside gShell. Use Relay to send welcome emails, notify Slack, or hit external webhooks whenever specific events occur.
